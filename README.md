# Resto Homepage

This is a homepage of Restaurant Website.

## Technologies
Project is created with:
* HTML5
* SASS version: ^4.13.0
* Swiper version: 5.3.0

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm install -g live-server
$ npm start
```

## Template
